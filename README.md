# go-crypto-ticker
Real-Time Crypto Ticker with InfluxDB


## Prerequisite(s)

[Task](https://taskfile.dev/)

```sh
brew install go-task/tap/go-task
```

## Build / Set up

* Start the stack:

```sh
task up
```

* Connect to [InfluxDB UI](http://localhost:8086/)

* Stop the stack:

```sh
task down
```
